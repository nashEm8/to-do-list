<h2>🎯  Objetivo</h2>
<p>O objetivo desse projeto foi criar um To-do list usando em maior parte javascript para sua construção e usuabilidade. Nele é possível remover um item da lista,
adicionar e também filtrar/buscar um item específico. Nessa aplicação usei conhecimentos como: funções, Manipulação de classes e do DOM, eventos, encadeamento de métodos
e refatoração do código, com o objetivo de deixá-lo mais limpo e compreensível para quem acessá-lo.
</p>
<br/><br/>
